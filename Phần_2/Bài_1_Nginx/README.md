# Bài 1: Chạy một container Nginx đơn giản

## Yêu cầu:
- Tạo một container chạy Nginx bằng Docker Compose
- Map cổng 8080 của máy host với cổng 80 của container

## Chạy bài tập:
```bash
docker-compose up -d
```

## Kiểm tra:
- Truy cập: http://localhost:8080

## Dừng:
```bash
docker-compose down
```
